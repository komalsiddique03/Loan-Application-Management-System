# 💰 Loan Eligibility Checking System

# 📖 Overview

The Loan Eligibility Checking System is a C++ console-based application developed to manage loan applications and determine whether an applicant is eligible for a loan based on predefined criteria such as age, salary, and requested loan amount.

The system also allows users to add, view, search, update, and delete loan applications while storing records in a text file for future use.

# ✨ Features

- 🔐 Password-based Login System
- ➕ Add New Loan Application
- ✅ Check Loan Eligibility
- 📋 View All Applications
- 🔍 Search Application by ID
- ✏️ Update Existing Application
- 🗑️ Delete Application
- 📊 View Application Statistics
- 💾 File Handling using `record.txt`
- 📁 Dynamic Memory Allocation
- 🎨 Colored Console Interface
- 🔔 Windows MessageBox and Beep Notifications

# 📌 Loan Eligibility Criteria

An applicant is considered **Eligible** if:

- Age is between **18 and 60 years**
- Salary is **50,000 or above**
- Loan amount is **1,000,000 or less**
Otherwise, the application is marked as **Not Eligible**.
